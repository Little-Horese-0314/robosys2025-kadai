#平均計算用コマンド
このコマンドは、標準入力から数値を読み取り、その平均を計算します。  
空行は無視されます  
数値以外の行は無視されます  
大量のデータでも簡単に平均値を求められます
#動作環境
Python 3.13.5

#使い方
1 スクリプトを実行可能にする
chmod +x average

2 ファイルから数値を読み取り、その平均を出す
(例) ./average < file.txt
